Bug Reports

When reporting bug reports, please keep in mind that Cevelop builds on Eclipse CDT. So if your problem is related to some core functionality and not one of our additions, please consider reporting the bug directly to Eclipse.

Reporting Cevelop Bugs

Please submit bug reports, feature wishesor other issues you might have to our GitHub Issue Tracker.
